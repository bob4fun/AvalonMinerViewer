Программа: AvalonMinerViewer
Актуальаня версия: 1.0.24.28
Разработчик: Бобров А.А.
Требования:
-ОС: Windows 7/8/10/11 РУ/EN + Net 4.8 Runtime-Desktop
-Масштаб экрана - любой

Группа в Телеграм: https://t.me/AvalonMinerViewer

Программа как есть не делает ни каких изменений в майнере, в расширенной версии позволяет управлять майнером посредством отправки команд.

Программа визуально оформляет данные с майнеров Avalon 10/11/12 серии полученные по API.

ВНИМАНИЕ: Визуальное отображение состояния чипа(ов) не является поводом для замены чипа или термопасты под радиаторами, 
а так-же не призывает к разборке майнера для каких либо необдуманных действий. 
Для определения степени неисправности и ремонта майнера необходимо обратиться в специализированный сервис.

Для корректной установки и работы программы, запускать её необходимо файлом setup.exe из папки Release. 
При запуске этим файлом установщик проверит соответствие системных требований.
